How to create access keys?
1. Go to cloud.tenable.io.
2. Access Settings from the sidebar
3. Find Account management section and select "My account"
4. Select API keys step in the vertical stepper
5. Click the generate button in the footer
6. In the dialog box, click generate again. Note: Existing keys will be overridden
7. API and Secret Keys will be generated. Note: Remember to copy API keys before leaving
